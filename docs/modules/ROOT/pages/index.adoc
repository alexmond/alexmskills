= alexmskills — Claude Code Skills Marketplace
:description: A curated Claude Code plugin marketplace of reusable, self-improving skills and agents — each an independently versioned plugin you install and update on demand.
:navtitle: Overview

`alexmskills` is a curated https://code.claude.com[Claude Code] **plugin marketplace** of reusable,
self-improving skills and agents. Each skill is packaged as an **independently versioned plugin**, so
you install and update exactly what you need.

The emphasis is on **self-learning** tooling — skills that get better at _your_ repository over time.

== The catalog

[cols="1,1,3",options="header"]
|===
| Plugin | Category | What it does

| xref:evolving-claude-md.adoc[evolving-claude-md] | self-learning
| Turns CLAUDE.md into a living Decisions & Learnings log that prunes, graduates, and archives itself via three hooks.

| xref:dev-crew.adoc[dev-crew] | self-learning
| A self-evolving, role-based delivery relay (architect → dev → qa → deployer), each role a subagent on its own model tier.

| xref:brainstorm-panel.adoc[brainstorm-panel] | self-learning
| Assembles a task-fit panel of role-specialized agents, picks a coordination style, and runs a generate-critique-refine loop.

| xref:learn-on-failure.adoc[learn-on-failure] | self-learning
| Auto-saves a durable learning to project memory whenever a task takes more than one fix cycle.

| xref:implement-issue.adoc[implement-issue] | workflow
| Drives a GitHub issue from branch → implement → verify → PR with a guided workflow.

| xref:maven-quality.adoc[maven-quality] | workflow
| Format, static-analysis, coverage, and pre-commit skills for Maven/Java projects.

| xref:security-audit.adoc[security-audit] | workflow
| Scans a codebase for OWASP-style vulnerabilities and reports them with remediation.

| xref:review-agents.adoc[review-agents] | review
| Read-only specialist subagents for code review, test running, and dependency/CVE auditing.

| xref:prompt-coach-beta.adoc[prompt-coach-beta] | beta
| Watches every prompt and nudges toward better prompting habits — 28 rules across 6 tiers, variant pool + progressive disclosure to avoid habituation, typo tolerance, and evidence-based praise for good behaviors.

| xref:research-sweep.adoc[research-sweep] | research
| Fans out independent research agents across distinct angles, then synthesizes and adversarially verifies.
|===
